# Procesamiento de Datos con Apache Spark y Kafka - Fase 3

Este proyecto presenta la implementación de soluciones de Big Data utilizando el ecosistema de Hadoop y Spark.

## 1. Análisis Batch con PySpark
Se realizó un análisis de datos sobre los beneficiarios en Colombia. Los datos fueron cargados en HDFS y procesados mediante Spark SQL/DataFrames para obtener estadísticas...

## 2. Procesamiento en Tiempo Real
Se implementó un flujo de datos continuo utilizando Apache Kafka como broker de mensajería y Spark Streaming para el consumo y procesamiento instantáneo de métricas...

## Tecnologías utilizadas:
* **Lenguaje:** Python (PySpark).
* **Almacenamiento:** HDFS (Hadoop Distributed File System).
* **Mensajería:** Apache Kafka 3.9.2.
* **Procesamiento:** Spark Streaming.
